# 🏎️ Motorsport Operations and Strategy Business Analysis

## 📊 Motorsport Dashboard

A multi-layered Power BI analytics solution designed to evaluate motorsport team performance across **Management**, **Engineering**, and **Operations** dimensions — all within a single integrated report file.

This project transforms race data into executive-ready insights through benchmarking, trend analysis, and strategic trade-off modeling.

# 🧠 Analytical Design Approach

The dashboard incorporates:

- Field-average benchmarking  
- Selected team dynamic filtering  
- Gap calculations (Selected vs Field)  
- Trade-off visualization  
- Multi-season trend evaluation  
- Quadrant-based strategic positioning  

All KPIs are structured using:

- Selected Team measures  
- Field Average measures  
- Gap measures  

This enables dynamic comparative intelligence for strategic decision-making.

---

# 🎨 Design Philosophy

The dashboard follows a structured executive layout:

- Minimalist interface  
- Clear KPI hierarchy  
- Consistent card-based design  
- Controlled color palette  
- Analytical clarity over decorative visuals

**Neutral Grey:** `#D8DCE2`

# 🏗️ Data Model Highlights

- Star-schema architecture  
- Dimensional modeling (Constructor, Race, Season)  
- KPI-specific views for Engineering, Management, and Operations  
- Custom DAX logic for:
  - Field benchmarking  
  - Gap analysis  
  - Dynamic comparative measures
